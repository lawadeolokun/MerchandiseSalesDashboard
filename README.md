# Merchandise Sales Dashboard

## Project Overview

This project analyses football merchandise sales data using Excel, PostgreSQL, and Tableau. The goal is to identify sales trends, top-performing products, regional performance, and key business insights through data analysis and interactive dashboards.

The project simulates a football club merchandise store selling products such as jerseys, scarves, hats, footballs, jackets, and training wear across multiple regions.

---

## Objectives

- Analyse merchandise sales performance.
- Identify top-selling products.
- Compare revenue across regions.
- Examine monthly sales trends.
- Create interactive dashboards for business decision-making.
- Demonstrate Excel, SQL, Tableau, and GitHub skills.

---

## Tools Used

- PostgreSQL

- GitHub

## Dataset

The dataset contains 1,000 merchandise sales transactions generated in Excel.

### Columns

| Column | Description |
|----------|-------------|
| OrderID | Unique order identifier |
| OrderDate | Date of purchase |
| Product | Product purchased |
| Category | Product category |
| Region | Sales region |
| Quantity | Quantity sold |
| UnitPrice | Price per unit (€) |
| Revenue | Quantity × Unit Price |

### Products

- Home Jersey
- Away Jersey
- Third Jersey
- Scarf
- Hat
- Football
- Training Top
- Jacket

### Categories

- Jerseys
- Accessories
- Equipment
- Training Wear

### Regions

- Dublin
- Cork
- Galway
- Limerick
- Waterford
- Belfast

---

## Data Preparation

The dataset was created and prepared in Microsoft Excel.

Data preparation steps included:

- Creating product lookup tables.
- Creating region lookup tables.
- Generating random order dates.
- Assigning products and regions.
- Calculating revenue values.
- Exporting the final dataset to CSV format.

---

## Key Insights

The dashboard enables users to:

- Identify the highest revenue-generating products.
- Compare regional sales performance.
- Analyse sales trends over time.
- Evaluate category performance.
- Monitor overall business performance using KPI metrics.
